经济与社会：理解性记忆笔记

说明：本笔记整理涉及专有名词、制度机制、概念辨析等需要理解性记忆的内容。
简单的三观判断、是非对错类题目也一并列出。
每个知识点下方用 note 标签展示相关真题，可查看每道题的逐选项分析。

第一章 经济学思维基础

1.0 课程介绍

了解性内容，记住关键人物即可。

【单选题】2014年诺贝尔经济学奖的获得者是（）。
A、阿比吉特·班纳吉
B、让·梯若尔 ✅
C、保罗·罗默
D、拉尔斯·皮特·汉森

选项分析：

  • A 阿比吉特·班纳吉：2019年诺贝尔经济学奖得主（减贫经济学），不是2014年。
  • B 让·梯若尔 ✅：法国经济学家，2014年因"对市场力量和监管的分析"获奖。他的核心贡献是对寡头垄断、平台经济学的研究。
  • C 保罗·罗默：2018年诺贝尔经济学奖得主（内生增长理论），不是2014年。
  • D 拉尔斯·皮特·汉森：2013年诺贝尔经济学奖得主（资产定价理论），不是2014年。

【单选题】经济学的研究在很长时间内与（）学科紧密结合。
A、数学 ✅
B、哲学
C、物理
D、化学

选项分析：

  • A 数学 ✅：经济学长期依赖数学建模（微积分、博弈论、优化理论等），从马歇尔到萨缪尔森，数学化是经济学发展的主线。
  • B 哲学：经济学确实起源于道德哲学（亚当·斯密是道德哲学教授），但"在很长时间内紧密结合"的学科是数学。
  • C 物理：物理学的方法论（如统计力学）对经济学有影响，但不是紧密结合的学科。
  • D 化学：与经济学没有直接紧密关系。

【判断题】著名经济学家让·梯若尔先生认为，有什么样的民众不一定有什么样的政策，二者没有必然联系。
答案：❌ 错误。梯若尔认为有什么样的民众就有什么样的政策，二者有必然联系。

1.1 约束最优化

沉淀成本（沉没成本）：以往已发生、与当前决策无关的费用。决策时应忽略沉淀成本，只看边际成本和边际收益。

经济学中的"约束"包括三类：

  • 预算约束：钱不够（收入有限）
  • 时间约束：时间不够（24小时有限）
  • 有限理性：认知能力不够（信息处理能力有限）

记忆口诀：钱、时、脑，三种约束。注意不包括"财富约束"。

【单选题】（）是指以往发生的，但与当前决策无关的费用。
A、机会成本
B、时间成本
C、沉默成本
D、沉淀成本 ✅

选项分析：

  • A 机会成本：为了选择某个方案而放弃的最佳替代方案的价值。比如你选了看电影就没法看书，看书的收益就是机会成本。注意：机会成本与"当前决策有关"，是决策时需要考虑的。
  • B 时间成本：花在某项活动上的时间所隐含的价值。不是"以往发生且与当前决策无关的费用"。
  • C 沉默成本：干扰项，没有这个经济学概念。是利用"沉没"与"沉默"的谐音来设置陷阱。
  • D 沉淀成本 ✅：即沉没成本（Sunk Cost），已经发生且无法收回的成本。比如买了一张电影票，不论你去看不去看，票钱都已花掉。决策时应忽略沉淀成本。

区分关键：沉淀成本 = 已经花了、收不回来的；机会成本 = 因为做了这个选择而放弃的那个最好选项。

【多选题】通常经济学中所说的约束是（）。ACD
A、预算约束 ✅
B、财富约束
C、时间约束 ✅
D、有限理性 ✅

选项分析：

  • A 预算约束 ✅：收入有限，不能无限消费。即"钱不够"。
  • B 财富约束 ❌：干扰项。经济学标准术语中没有"财富约束"这个概念。注意区分"预算约束"（收入流量的约束）和"财富"（存量概念），考试中不能混淆。
  • C 时间约束 ✅：每天只有24小时，时间有限。即"时间不够"。
  • D 有限理性 ✅：人的认知能力有限，无法处理所有信息。即"脑力不够"。这是西蒙（Herbert Simon）提出的概念。

记忆：B"财富约束"是干扰项。经济学中三种约束 = 预算约束 + 时间约束 + 有限理性。

1.2 个人理性 vs 集体理性

核心逻辑：亚当·斯密认为人是"自私自利"的经济人，但"看不见的手"（市场机制）可以将个人自利行为引导为公共利益。

关键辨析：

  • “看不见的手” = 市场的调节作用（不是政府）
  • “看得见的手” = 政府的干预调控（不是市场）
  • “看不见的手搞不定"不代表"看得见的手一定能搞定”——政府也会失灵

囚徒困境：本质是集体理性 vs 个人理性的冲突。每个人都做对自己最优的选择，结果反而对所有人都不好。

【判断题】亚当·斯密在《国富论》中有一个基本的假设：人都是"自私自利"的经济人。（）
答案：✅ 正确。

【判断题】"看不见的手"搞不定的事情，"看得见的手"一定可以搞定。（）
答案：❌ 错误。市场失灵不代表政府一定行。政府也可能失灵（如信息不对称、寻租、官僚低效等）。

【判断题】看得见的手是指市场的调节作用。（）
答案：❌ 错误。

  • 看不见的手 = 市场（价格机制自动调节供需）
  • 看得见的手 = 政府（政策干预、宏观调控）

注意别搞反了！"看得见的手"是政府，"看不见的手"是市场。

1.3 激励效应

激励效应：理性人不会以静止观点看问题——人会根据激励改变行为。政策制定要考虑激励效应，否则好心办坏事。

【单选题】理性人不会以静止的观点看问题，指的是（）。
A、激励效应 ✅
B、外溢效应
C、内化效应
D、动态效益

选项分析：

  • A 激励效应 ✅：人会根据激励改变行为。比如对安全带的法律要求，原意是保护司机，但司机因为有了安全带反而开得更快（风险补偿行为）。理性人不是静止的，会根据制度/政策调整自己的行为。
  • B 外溢效应（溢出效应）：某个经济主体的行为对其他主体产生的外部影响（正或负），但行为人自己不承担后果。如工厂排污影响周边居民。这是外部性概念，不是"不以静止观点看问题"。
  • C 内化效应：将外部成本或收益转化为私人成本或收益的过程。如对排污企业征税，使其承担社会成本。这是解决外部性的手段。
  • D 动态效益：经济系统中随时间变化产生的效率提升。这是一个较宽泛的概念，不是本题所指。

关键理解：激励效应的核心是——“人会变”。你不能假设政策出台后人还跟之前一样行事。

【单选题】根据斯蒂芬·考维在《高效人士的七个习惯》中的理论，重要但不紧急属于第（）等级。
A、一
B、二 ✅
C、三
D、四

选项分析：

  • 第一象限：重要且紧急（救火、危机处理）
  • 第二象限 ✅：重要但不紧急（长期规划、学习、锻炼、关系建设）——这是考维认为最应该投入时间的象限
  • 第三象限：不重要但紧急（电话打扰、某些会议）
  • 第四象限：不重要且不紧急（刷手机、闲聊）

虽然这不是经济学概念，但考维的理论在本课中出现，需记住"重要但不紧急 = 第二等级"。

【单选题】《中国的经济制度》一书出自（）。
A、郎咸平
B、吴敬琏
C、厉以宁
D、张五常 ✅

选项分析：

  • A 郎咸平：香港中文大学教授，以"郎监管"闻名，曾批评国企改革中的国有资产流失问题。
  • B 吴敬琏：被称为"吴市场"，主张市场经济改革的经济学家。
  • C 厉以宁：被称为"厉股份"，中国股份制改革的倡导者。
  • D 张五常 ✅：新制度经济学代表人物，科斯的学生。《中国的经济制度》是他对中国经济改革（尤其是县级竞争制度）的经济学分析。

记忆：吴敬琏=“吴市场”，厉以宁=“厉股份”，张五常=《中国的经济制度》

【判断题】学习经济学需要具备企业家精神。（）
答案：✅ 正确。经济学中的企业家精神指对市场机会的敏锐洞察、风险承担和创新能力，学习经济学也需要这种探索精神。

1.4 非理性行为

【多选题】以下属于非理性行为的有（）。ABC
A、选择困难症 ✅
B、过度自信 ✅
C、拖延症 ✅
D、“抓大放小”

选项分析：

  • A 选择困难症 ✅：面对多个选项时无法做出决策，违反了经济学中"理性人能在约束条件下最优化选择"的假设。
  • B 过度自信 ✅：高估自己的能力或判断准确性。经济学中，过度自信会导致投资者过度交易、企业过度扩张等。
  • C 拖延症 ✅：明知应该做某事但一再推迟。行为经济学称之为"现时偏误"（present bias）——过度看重眼前的舒适而忽视长远利益。
  • D “抓大放小” ❌：这反而是理性的决策策略——在资源有限时聚焦最重要的事（抓住主要矛盾），这恰恰体现了经济学中的约束最优化思维。

【判断题】买白菜很计较，买股票很冲动，这是一种非理性行为。（）
答案：✅ 正确。这就是标度效应的体现——在金额小的事情上精打细算，在金额大的事情上反而草率决策。因为人对不同量级事物的感知精度不同。

1.5 神经科学与决策

边际效应：经济学与心理学的结合。人对变化的感知是边际的，不是绝对的。

多巴胺神经元：

  • 主管评价体系（报酬预测）
  • 受损会导致"报酬预测错误"
  • 多巴胺与**“需要”**相关（不是"喜欢"，也不是"欢愉"）
  • 多巴胺是对**"超预期"变化**做出反应

韦伯定律：只有当两件物品的重量差异超过一个恒常比例，人们才能感知到这种差异。→ 本质是相对感知，不是绝对感知。

资本市场同理：市场只对"超预期"做出强烈反应，符合预期的消息不会引起大的波动。

【单选题】边际效应是经济学和（）的紧密结合。
A、数学
B、心理学 ✅
C、逻辑学
D、政治学

选项分析：

  • A 数学：经济学确实大量使用数学，但边际效应特指经济学与心理学的交叉领域。
  • B 心理学 ✅：边际效应研究人对"增量变化"的心理感知，是行为经济学/神经经济学的核心概念。
  • C 逻辑学：逻辑学提供推理工具，但不是边际效应的研究伙伴。
  • D 政治学：政治学关注权力与治理，与边际效应无直接关联。

【单选题】（）神经元主管我们的评价体系，如果受损会导致"报酬预测错误"。
A、谷氨酸
B、胆碱
C、多巴胺 ✅
D、GABA

选项分析：

  • A 谷氨酸：大脑中最主要的兴奋性神经递质，参与大部分脑区信号传递，但不是专门管"报酬评价"的。
  • B 胆碱（乙酰胆碱）：与记忆和学习相关（阿尔茨海默症就与胆碱系统退化有关），不是报酬评价体系。
  • C 多巴胺 ✅：主管报酬预测和评价体系。多巴胺神经元会对"超预期"的奖赏做出强烈反应，对"符合预期"的不反应，对"低于预期"的反而抑制。受损会导致无法正确预测和评价报酬。
  • D GABA（γ-氨基丁酸）：大脑中最主要的抑制性神经递质，起"刹车"作用，与报酬评价无关。

【单选题】肯特·贝利齐实验的结论是多巴胺神经元与（）相关。
A、欢愉
B、喜欢
C、需要 ✅
D、厌恶

选项分析：

  • A 欢愉：多巴胺不是欢愉的物质。实际上，内啡肽等才与欢愉感直接相关。多巴胺更多是驱动"追求"而非"享受"。
  • B 喜欢：多巴胺 ≠ 喜欢。你可以"需要"但并不"喜欢"（如成瘾者需要毒品但可能已经不再"喜欢"它）。
  • C 需要 ✅：贝利齐（Berridge）的实验区分了"想要"（wanting/需要）和"喜欢"（liking）。多巴胺系统对应的是"需要/想要"，是一种驱动力。
  • D 厌恶：与多巴胺无关。

关键区分：多巴胺 = 需要（wanting）≠ 喜欢（liking）≠ 欢愉（pleasure）

【单选题】（），即只有当两件物品重量差异超过一个恒常比例，人们才能准确感知到这种重量差异。
A、巴莱特定律
B、科斯定律
C、墨菲定律
D、韦伯定律 ✅

选项分析：

  • A 巴莱特定律（二八定律/帕累托法则）：80%的结果来自20%的原因。如80%的销售额来自20%的客户。与感知无关。
  • B 科斯定律（科斯定理）：在交易成本为零时，产权的初始配置不影响资源配置效率。这是制度经济学概念，与感知无关。
  • C 墨菲定律：凡是可能出错的事，一定会出错。这是一种概率思维，与感知无关。
  • D 韦伯定律 ✅：德国心理学家韦伯发现，人的感觉差异取决于相对比例而非绝对差值。比如从1kg增加到1.1kg你能感觉到，但从10kg增加到10.1kg你可能感觉不到——虽然都是增加了0.1kg。

韦伯定律的核心：感知取决于相对变化比例，不是绝对变化量。

【判断题】经济学研究中把人设置成完全理性是不可取的。（）
答案：✅ 正确。人有认知偏差、情绪影响、有限理性等，完全理性的假设过于简化。

【判断题】经济学中所认为的非理性，其实就是大脑认可的报酬系统与社会认可的报酬系统不一致。（）
答案：✅ 正确。大脑的多巴胺系统追求即时满足，社会规范要求长远规划，两者不一致就产生"非理性"。

【判断题】多巴胺通常被认为是对"超预期"变化做出反应。（）
答案：✅ 正确。多巴胺不是对奖赏本身反应，而是对"比预期更好"的部分反应。

【判断题】资本市场通常只对"超预期"做出强烈反应。（）
答案：✅ 正确。好消息如果已经被市场预期到了，股价不会涨；只有"超出预期"的消息才会引起剧烈波动。

1.6 标度效应

标度效应：生活中很多非理性行为的根本原因。人对不同量级事物的感知精度不同。

核心机制：改变参考点 = 以精度换尺度。参考点不同，感受完全不同（“战略上藐视，战术上重视” = 参考点切换）。

关键辨析：标度效应下，感知的相对量变化差异很大（不是"绝对量变化相同"）。

【单选题】生活中的很多非理性行为的本质原因是（）。
A、马太效应
B、标度效应 ✅
C、蝴蝶效应
D、同伴效应

选项分析：

  • A 马太效应："强者越强，弱者越弱"的累积优势现象。如富人更容易变得更富。这不是"非理性行为"的原因，而是一种社会现象。
  • B 标度效应 ✅：人对不同量级（尺度）事物的感知精度不同。比如你在超市省了5块钱很高兴，但买房时对5万块的差异反而不敏感。量级越大，感知精度越低，导致非理性行为。
  • C 蝴蝶效应：混沌理论中的概念——初始条件的微小差异会导致结果的巨大不同。与人的非理性行为无关。
  • D 同伴效应：人的行为受到周围人的影响（如"别人都买所以我也买"）。这是羊群效应，但不是"很多非理性行为的本质原因"。

【单选题】以有限应对无限的方式是（）。
A、改变思维
B、改变视角
C、改变条件
D、改变参考点 ✅

选项分析：

  • A 改变思维：太笼统，不是经济学的精准表述。
  • B 改变视角：接近但不精确。
  • C 改变条件：指改变外部约束，不是本题答案。
  • D 改变参考点 ✅：标度效应的核心应对策略——通过切换参考点（比较基准），以精度换尺度。比如买10万的车时不要想着"才多1000块"，而要想想1000块在日常生活中能买什么。

【判断题】标度效应下，改变参考点乃是以精度换尺度。（）
答案：✅ 正确。切换到小的参考点，你恢复了对细节的感知精度，但失去了对大局的把握（尺度）。

【判断题】标度效应下，感知的绝对量变化相同，但相对精度却差异很大。（）
答案：❌ 错误。注意题目说的是"绝对量变化相同"——这是错的。标度效应下，是相对量变化差异大，不是绝对量变化相同。比如从1kg到2kg（变化100%）你能感知，从100kg到101kg（变化1%）你感知不到。变化的绝对量不同，感知也不同。

辨析关键：标度效应下，感知差异取决于相对比例（参考韦伯定律），不是绝对量相同。

【判断题】“战略上藐视敌人，战术上重视敌人”，这实际是参考点的切换问题。（）
答案：✅ 正确。“战略上藐视” = 用大尺度参考点看问题（整体看没那么可怕），“战术上重视” = 用小尺度参考点看问题（每个细节都要认真对待）。

A、婚姻 ✅
B、家庭 ✅

第二章 劳资关系

2.1 委托-代理问题

泰勒制：流水线监督工人 + 产品标准化的生产模式。核心思想是将劳动过程标准化，减少工人自主权。

现代企业的根本矛盾——两权分离：所有权（股东）与经营权（管理层）分离，导致：

  • 目标差异：委托人（股东）和代理人（管理层）的目标不一致
  • 信息不对称：代理人拥有委托人不了解的私人信息

注意：代理人（管理者）不一定拥有企业所有权。代理人是被委托人雇佣来经营的。

【单选题】让流水线监督工人，产品必须标准化，这种生产模式实际是（）。
A、泰勒制 ✅
B、沃伦制
C、人才制
D、雇佣制

选项分析：

  • A 泰勒制 ✅：由弗雷德里克·泰勒（Frederick Taylor）提出的"科学管理"理论。核心：将工作分解为标准化动作，通过流水线监督工人执行，消除工人的自主判断空间，提高效率。本质是用标准化解决信息不对称问题。
  • B 沃伦制：干扰项，没有这个经济学/管理学概念。
  • C 人才制：干扰项，不是标准术语。
  • D 雇佣制：泛指雇佣劳动关系，太过宽泛，不特指"流水线+标准化"的生产模式。

【多选题】现代企业，所有权与经营权的两权分离，造成委托——代理关系中的代理问题有（）。AB
A、目标差异 ✅
B、信息不对称 ✅
C、无合同保障
D、经营权高于所有权

选项分析：

  • A 目标差异 ✅：股东（委托人）追求企业价值最大化，管理层（代理人）可能追求自身利益（如薪酬、权力、在职消费）。双方目标不一致。
  • B 信息不对称 ✅：管理层比股东更了解企业的日常经营状况，可以利用信息优势为自己谋利。
  • C 无合同保障 ❌：现代企业通常有完善的合同体系（薪酬合同、股权激励等），"无合同保障"不是代理问题的本质。
  • D 经营权高于所有权 ❌：干扰项。所有权始终高于经营权（股东是企业的所有者），只是两权分离后所有者对经营者的控制力减弱了，但并非"经营权高于所有权"。

【判断题】企业代理人一定拥有企业所有权。（）
答案：❌ 错误。代理人是被雇佣来经营企业的人（如CEO、管理层），他们不一定是企业的所有者（股东）。当然，有些代理人也可能持有部分股份，但这不是"一定"的。

2.2 信息不对称的两种类型

类型 含义 别名 发生时机
隐藏类型 一方不知道对方的"属性"（如能力高低） 逆向选择 缔约前
隐藏行动 一方无法观测对方的"行为"（如是否努力） 道德风险 缔约后

关键记忆：道德风险 = 隐藏行动的非对称信息。缔约前代理人先获知自己的类型。

【单选题】道德风险也称为（）。
A、隐藏类型的非对称信息
B、隐藏行动的非对称信息 ✅
C、隐藏类型的信息
D、隐藏行动的信息

选项分析：

  • A 隐藏类型的非对称信息：这是逆向选择，不是道德风险。隐藏类型 = 缔约前一方不了解对方的属性（如保险公司不了解投保人的健康状况）。
  • B 隐藏行动的非对称信息 ✅：这就是道德风险（Moral Hazard）。缔约后，委托人无法观测代理人是否努力工作。比如买了保险后开车更鲁莽。
  • C 隐藏类型的信息：表述不完整，缺少"非对称"三字。标准术语需要强调"非对称信息"。
  • D 隐藏行动的信息：表述不完整，缺少"非对称"三字。

区分关键：道德风险 = 隐藏行动（缔约后）| 逆向选择 = 隐藏类型（缔约前）

【单选题】以下属于缔约过程事前的是（）
A、委托人提供合同菜单
B、代理人接受或拒绝
C、代理人获知自己类型 ✅
D、代理人选择行动

选项分析：

  • A 委托人提供合同菜单：这是委托人的行动，属于缔约过程中的中间环节。
  • B 代理人接受或拒绝：发生在委托人提供合同之后。
  • C 代理人获知自己类型 ✅：在缔约之前，代理人就知道自己的能力、偏好等属性。这是"事前"环节——代理人先知道自己是什么类型，然后才缔约。
  • D 代理人选择行动：这是缔约之后的行为，属于"事后"环节。

缔约过程的时间线：代理人获知自己类型（事前）→ 委托人提供合同 → 代理人接受/拒绝 → 代理人选择行动（事后）

【判断题】伴君如伴虎，体现了经济学中的代理人有私人信息。（）
答案：❌ 错误。“伴君如伴虎"体现的是委托人（皇帝/君主）有私人信息——你不知道皇帝什么时候会翻脸，是委托人（君主）的不可预测性让代理人（臣子）感到恐惧。题目说的是"代理人有私人信息”，搞反了角色。

实际上这道题可以从另一个角度理解：在古代君臣关系中，皇帝是委托人，臣子是代理人。但"伴君如伴虎"的恐惧来自皇帝（委托人）行为的不确定性，而不是代理人（臣子）的信息优势。

2.3 效率工资

核心逻辑：老板的激励难题需要将合同订立在某种可观测的"绩效指标"上。

效率工资成立条件：

  1. 好业绩和差业绩的工资差要足够大（胡萝卜+大棒）
  2. 将工资提高到竞争性水平之上，形成可观的**“失业大军”** → 失业威胁才有效

推论：固定工资 + 努力无助于业绩 = 员工必然偷懒。

【单选题】效率工资下，老板的激励难题需要将合同订立在某种（）"绩效指标"上。
A、可执行
B、可观测 ✅
C、可分析
D、可对比

选项分析：

  • A 可执行：合同能不能执行是法律层面的问题，不是解决激励问题的关键。
  • B 可观测 ✅：核心在于老板能看到员工的表现。如果业绩无法被观测，就无法据此奖惩。比如论文发表数量是可观测的，而员工的"努力程度"本身不可直接观测，需要转化为可观测的指标。
  • C 可分析：分析是事后的事，关键在于能否被观测到。
  • D 可对比：对比需要，但前提是先要可观测。

记忆：效率工资的关键 = 把合同建立在可观测的绩效指标上。

【判断题】通过效率工资的博弈阐释，我们可以推论出固定工资，或者努力无助于业绩时，员工必然偷懒。（）
答案：✅ 正确。固定工资意味着无论努力还是偷懒，收入都一样 → 理性的员工必然选择偷懒。或者即使有绩效工资，但如果努力对业绩没有帮助（如业绩完全由运气决定），员工也不会努力。

【判断题】只有好业绩和差业绩的工资差足够大，效率工资才能得以成立。（）
答案：✅ 正确。如果工资差太小，偷懒被发现后的损失不大，员工就没有动力努力工作。必须"胡萝卜够甜、大棒够狠"。

【判断题】当企业将工资水平提高到"竞争性"水平之上时，数量可观的"失业大军"对在岗员工影响有限。（）
答案：❌ 错误。失业大军对在岗员工的影响很大，这正是效率工资的机制——因为失业的威胁很大（外面有很多失业者在等着），在岗员工才不敢偷懒，因为被解雇的代价很高。

【判断题】现实情况中，单纯的"高薪"无法"养廉"。（）
答案：✅ 正确。高薪只是效率工资的一个条件，还需要配合有效的监督机制（可观测的绩效指标）和足够的惩罚力度。

2.4 标尺竞争

定义：通过比较相似代理人的业绩来进行激励。

成立条件（限制）：

  • 各代理人面临共同冲击（面对相似的外部环境）
  • 各代理人业绩可比
  • 各代理人勤勉对业绩促进功效相若（努力的效果相似）

【单选题】经济学中的"囚徒困境"指（）之间的冲突。
A、集体利益与个人利益
B、集体损失与个人损失
C、集体理性与个人理性 ✅
D、集体期望与个人期望

选项分析：

  • A 集体利益与个人利益：看似有道理，但"利益"是静态的结果描述，不如"理性"精准。
  • B 集体损失与个人损失：囚徒困境的双方都受损，但"损失"不是对冲突本质的描述。
  • C 集体理性与个人理性 ✅：囚徒困境的本质——从个人理性出发（坦白对我最有利），导致了对集体理性来说最差的结果（两人都坦白，刑期最长）。注意：是"理性"对"理性"的冲突，不是"利益"对"利益"的冲突。
  • D 集体期望与个人期望：期望不是经济学中的标准表述。

关键：囚徒困境 = 个人理性与集体理性的冲突。每个人都做对自己最优的选择，结果对所有人都不好。

【多选题】标尺竞争的限制包括（）。ABD
A、各代理人面临共同冲击 ✅
B、各代理人业绩可比 ✅
C、各代理人之间的竞争存在不公平
D、各代理人勤勉对业绩促进功效相若 ✅

选项分析：

  • A 共同冲击 ✅：各代理人必须面对相似的外部环境。如果A在经济繁荣区，B在经济衰退区，直接比较业绩就不公平。
  • B 业绩可比 ✅：业绩的衡量标准要一致。不能一个考核销售额，另一个考核客户满意度。
  • C 竞争存在不公平 ❌：不公平是标尺竞争要避免的问题，不是它的"限制条件"。标尺竞争的前提恰恰是竞争要公平。
  • D 勤勉对业绩促进功效相若 ✅：如果A特别努力能大幅提升业绩，而B再怎么努力业绩也提升不了多少，那比较A和B的业绩就没有意义。

2.5 GDP锦标赛

政治企业家 vs 经济企业家：

  • 目标更复杂（不是更单一）
  • 可动用的资源更多（不是更少）

关键判断：

  • GDP锦标赛中，招商引资对他人GDP也有影响（不是"影响有限"）
  • GDP已不能作为充分统计量

【多选题】相比经济企业家，政治企业家的特点是（）。AC
A、目标更复杂 ✅
B、目标更单一
C、可以动用的资源更多 ✅
D、可以动用的资源更少

选项分析：

  • A 目标更复杂 ✅：经济企业家的目标相对单一（利润最大化），政治企业家需要兼顾经济增长、社会稳定、民生等多重目标。
  • B 目标更单一 ❌：与A矛盾，政治企业家的目标是更复杂，不是更单一。
  • C 可动用资源更多 ✅：政治企业家掌握政府资源（财政、土地、政策等），比经济企业家的资源更丰富。
  • D 可动用资源更少 ❌：与C矛盾。

【判断题】在中国的GDP竞标赛中，招商引资可以做高自己的GDP，对他人GDP影响有限。（）
答案：❌ 错误。招商引资（如引进一个大企业）不仅影响本地GDP，还会通过产业链、劳动力流动等渠道影响其他地区的GDP。GDP之间存在外部性。

【判断题】目前的中国经济下，GDP仍然可以作为充分统计量。（）
答案：❌ 错误。GDP不能反映环境污染、收入分配、居民幸福感等重要维度，已不能作为衡量经济发展质量的"充分统计量"。

第三章 择业与职业发展

3.1 人力资本分类

人力资本 物质资本
通用性 在各种企业都能发挥作用的技能 容易在市场中变现的机器设备
专用性 只能在特定企业发挥作用的技能 很难在市场中变现的机器设备

沉淀成本 ≠ 生产启动成本。沉淀成本是已经投入且无法收回的成本。

【多选题】以下关于资本分类的说法，正确的是（）。ABCD
A、通用性人力资本是指在各种企业都能发挥作用的专业技能 ✅
B、专用性人力资本是指只能在特定企业发挥作用的专业技能 ✅
C、通用性物质资本是指容易在市场中变现的各种机器设备 ✅
D、专用性物质资本是指很难在市场中变现的各种机器设备 ✅

选项分析：

  • A 通用性人力资本 ✅：如Excel技能、英语能力、编程能力——无论在哪家公司都有用。
  • B 专用性人力资本 ✅：如只适用于某公司内部系统的操作技能、只与某个老板配合的工作方式——离开了这个特定环境就没用了。
  • C 通用性物质资本 ✅：如通用设备、标准化的机器——很容易在二手市场卖掉。
  • D 专用性物质资本 ✅：如为某公司定制的生产线——很难转卖给其他企业。

核心区分：通用性 = 到哪都有用 | 专用性 = 离开就没用

【判断题】沉淀成本是生产需要预支的启动成本。（）
答案：❌ 错误。沉淀成本是已经投入且无法收回的成本，不是"预支的启动成本"。启动成本是可以带来未来收益的投资，而沉淀成本是沉没了（sunk）的、不可回收的。

【判断题】如果从经济学出发，相比于不会游泳的人，会游泳的人更容易淹死。（）
答案：✅ 正确。这是激励效应的体现。会游泳的人因为自信而更可能去深水区/危险水域，承担更多风险（风险补偿行为）。

3.2 人力资本积累的两个问题

  • 总量问题：花多少时间学习 vs 多少时间娱乐
  • 结构问题：花多少时间在通用性 vs 专用性人力资本上
    • 结构决定你与老板分割蛋糕的比例（议价能力）
    • 结构也决定你与老板能做多大蛋糕

拍马屁 = 积累通用性人力资本（不是专用性）。因为拍马屁的技能在任何企业都能用。

【单选题】关于员工如何积累自己的人力资本，以下说法错误的是（）。
A、人力资本积累的总量问题是花多少时间学习，多少时间娱乐
B、人力资本积累的结构问题是花多少时间在通用性人力资本，多少在专用性人力资本
C、人力资本积累的结构问题不能决定与老板可以分享的蛋糕到底能有多大 ✅（此选项错误）
D、人力资本积累的结构问题可以决定与老板分割蛋糕时可以占有多大比例。

选项分析：

  • A 总量问题 ✅（说法正确）：学习时间vs娱乐时间的分配。
  • B 结构问题 ✅（说法正确）：通用性vs专用性人力资本的分配。
  • C ✅（此选项错误，所以是答案）：结构问题能够决定蛋糕的大小。因为专用性人力资本让员工与特定企业更匹配，能创造更大的价值。说"不能决定"是错的。
  • D ✅（说法正确）：通用性人力资本多 → 外部机会多 → 议价能力强 → 分割蛋糕比例有利。

结构问题的两个维度：① 决定蛋糕能做多大（匹配效率）② 决定分蛋糕时能占多少比例（议价能力）

【判断题】从经济学出发，与拍马屁相比，钻业务是积累专用性人力资本。（）
答案：❌ 错误。拍马屁（人际关系、向上管理）在任何企业都能用 → 通用性人力资本。钻业务（深入了解某公司的特定流程和系统）→ 专用性人力资本。

注意区分：拍马屁 = 通用性（到哪都能拍），钻业务 = 专用性（换个公司业务就不同了）。

3.3 非升即走制度（Up or Out / Tenure-track）

典型实例：律师事务所的合伙人制度、高校tenure-track制度。

核心逻辑链：

预备期应对科研进行强激励（不是弱激励），因为论文发表容易观察，教学绩效难以度量。

双边道德风险：

  • 员工风险：努力获得专用性人力资本后，公司可能无故解雇
  • 引入非升即走后，留下的员工必须给高工资（公司承诺不解雇）

【单选题】以下属于"非升即走"制度的是（）。
A、公务员制度
B、律师事务所的合伙人制度 ✅
C、快递行业制度
D、中学教师制度

选项分析：

  • A 公务员制度：公务员是"铁饭碗"，进来后基本不会被淘汰，没有"or out"的部分。
  • B 律师事务所的合伙人制度 ✅：初级律师经过考核期后，要么晋升为合伙人（up），要么离开（out）。典型的"非升即走"。
  • C 快递行业制度：没有明确的"升/走"考核机制。
  • D 中学教师制度：一般没有严格的"非升即走"机制（不像高校tenure-track）。

【判断题】高校里面的tenure-track制度是"非升即走"的表现。（）
答案：✅ 正确。助理教授经过6-7年的预备期考核，要么获得终身教职（tenure），要么离开。

【判断题】新劳动法规定，连续两个合同期之后，变成不定期合同。这一规定事实上不利于劳动者的权益。（）
答案：❌ 错误。这一规定有利于劳动者——提高了劳动保护，增加了解雇成本，使劳动者更难被随意解雇。从劳动者权益角度看是有利的（虽然从市场效率角度可能有争议）。

【单选题】在"非升即走"制度下，以下有关高校中科研与教学的说法错误的是（）。
A、论文发表容易观察
B、教学绩效难以度量
C、非升即走能够筛选出高科研能力教师
D、预备期应对科研进行弱激励 ✅（此选项错误）

选项分析：

  • A 论文发表容易观察 ✅（说法正确）：论文数量、期刊等级、引用次数都是客观可观测的指标。
  • B 教学绩效难以度量 ✅（说法正确）：教学效果（学生真的学到了什么）很难量化。
  • C 非升即走筛选高科研能力者 ✅（说法正确）：预备期考核主要看科研成果，能通过的人科研能力较强。
  • D ✅（此选项错误，所以是答案）：预备期应该对科研进行强激励（不是弱激励），因为论文容易观察且易于考核，应该加大科研考核力度。

【判断题】企业中，引入"非升即走"制度，并不能解决应聘者的逆向选择问题。（）
答案：❌ 错误。非升即走可以解决逆向选择问题——通过预备期的严格考核（筛选效应），低能力者会被淘汰，高能力者会留下，从而解决了"不知道应聘者能力高低"的逆向选择问题。

【判断题】高校中，终身教职为老员工的职位安全提供保证，以此也可为老员工提供合适的招聘激励。（）
答案：✅ 正确。终身教职让老员工不用担心因年龄增长被解雇，从而愿意帮助招聘和培养新人。

【单选题】在企业中，引入"非升即走"制度后，下列关于双边道德风险的说法，错误的是（）。
A、专用性人力资本能够提高公司绩效
B、员工努力之后更有可能获得高的专用性人力资本
C、员工即使努力获得了较高的专用性人力资本，公司也会无故解雇员工。 ✅（此选项错误）
D、引入非升即走之后，预备期满的员工留下来，就必须给出高工资

选项分析：

  • A ✅（说法正确）：专用性人力资本让员工更了解企业、配合更默契 → 提高绩效。
  • B ✅（说法正确）：努力 → 积累更多专用性人力资本。
  • C ✅（此选项错误，所以是答案）：引入非升即走制度后，通过预备期的员工获得终身教职保障，公司不会无故解雇。这正是非升即走解决双边道德风险的机制——保护员工不被"过河拆桥"。
  • D ✅（说法正确）：留下的员工必须给高工资，作为公司承诺不解雇的"补偿"。

【单选题】以下有关"彼得原理"的说法，错误的是（）。
A、在绝大多数的组织里，每个人最终都会停在自己能胜任的职位上 ✅（此选项错误）

选项分析：

  • A ✅（此选项错误，所以是答案）：彼得原理说的是每个人最终停在自己不能胜任的职位上（不是"能胜任"）。因为你在某个职位做好了 → 被晋升 → 直到晋升到一个你做不好的位置 → 停下来。
  • B ✅（说法正确）：小材大用 = 能力不匹配岗位 → 组织绩效受损。
  • C ✅（说法正确）：非升即走强制淘汰不合格者，让更有能力的人上去。
  • D ✅（说法正确）：庸者下、贤能者上是提高绩效的正确方向。

【判断题】彼得原理是指在各种组织中，由于习惯于对在某个等级上称职的人员进行晋升提拔，因而雇员总是趋向于被晋升到其不称职的地位。（）
答案：✅ 正确。这是彼得原理的标准定义。

【判断题】企业引入"非升即走"制度后，在一定时间内无法打开局面者，将更有可能是该职务层面的低能力者。（）
答案：✅ 正确。预备期内无法做出成绩 → 很可能是能力不足 → 被淘汰。

3.4 新劳动法的影响

两个合同期后签不定期合同 = 提高了解雇成本：

  • 好处：提高了劳动保护
  • 坏处：劳动力市场变僵化
  • 反直觉：职位稳定反而不利于积累专用性人力资本（没有竞争压力）

第四章 科斯定理与企业边界

4.1 科斯定理

科斯定理：在交易成本为零时，产权的初始界定不影响资源配置效率——资源的有效使用与初始产权的界定无关。

企业的本质：用指令替代价格机制。企业内部通过管理指令协调，而非市场价格协调。

关键辨析：科斯定理说的是"初始产权界定无关"，不是说"产权不重要"。

【单选题】科斯定理表明资源的有效使用，与初始产权的界定（）。
A、正相关
B、负相关
C、无关 ✅
D、不确定关系

选项分析：

  • A 正相关：错误。科斯定理的核心结论是无关，不是正相关。
  • B 负相关：错误。
  • C 无关 ✅：科斯定理——当交易成本为零时，无论产权初始归谁，市场交易都能达到有效配置。比如上游工厂排污权归谁，只要交易成本为零，最终污染治理的效率是一样的。
  • D 不确定关系：不是"不确定"，而是明确的"无关"。

记忆：科斯定理 = 交易成本为零时，初始产权界定不影响资源配置效率。

【判断题】企业的本质是用指令替代价格机制。（）
答案：✅ 正确。科斯在《企业的性质》中提出，企业之所以存在，是因为用管理指令协调比用市场价格协调更有效率（在一定范围内）。

4.2 企业的边界与规模

企业规模的两面性：

  • 规模越大 → 管理效率下降，信息渗漏和扭曲加重
  • 规模越小 → 指令运行越通畅
  • 互联网冲击下，"中间规模"企业处于两难境地——大而强或小而美，中间规模"两不靠"

关键判断：企业的规模越大，指令运行越没有效（不是越有效）。

【单选题】关于企业的性质，以下说法正确的是（）。
A、企业规模增加，指令配置资源的交易成本逐渐递减。
B、企业的规模越小，指令运行越没有效率。
C、互联网出现对企业规模没有影响。
D、互联网冲击下，"中间规模"企业处于两难境地。 ✅

选项分析：

  • A 错误：企业规模增加，指令配置资源的交易成本逐渐递增（不是递减），因为管理层次增多、信息扭曲加重。
  • B 错误：企业的规模越小，指令运行越通畅（不是"没有效率"）。
  • C 错误：互联网对企业规模有重大影响（促进分工、降低协调成本）。
  • D ✅：互联网使得大企业更强大（平台效应），小企业更灵活（长尾市场），而中间规模的企业两头不靠。

【判断题】企业规模增加，会导致管理效率下降，加重信息渗漏和扭曲。（）
答案：✅ 正确。

【判断题】企业的规模越大，指令运行越有效。（）
答案：❌ 错误。企业规模越大，管理层级越多，信息传递中的渗漏和扭曲越严重，指令运行效率反而下降。

4.3 产权与权威

产权的双重意义：

  • 经济意义上的产权：实际控制和使用资源的能力
  • 法律意义上的产权：法律确认和保护的权利

注意：既存在经济意义上的产权，又存在法律意义上的产权——两者都存在。

股权分散的现代大企业：所有者（股东）失去对管理层（代理人）的权威。

不平等投票权：可以解决股权分散的问题（如AB股结构，创始人的股票投票权更高）。

【单选题】关于产权与权威的概念，以下正确的是（）。
A、产权不是一种真正的权利。
B、产权分为收益权和转让权。
C、既存在经济意义上的产权，又存在法律意义上的产权。 ✅
D、权威与产权关系不大。

选项分析：

  • A 错误：产权是真实的权利。
  • B 错误：产权通常包括使用权、收益权、转让权等，不只是"收益权和转让权"。
  • C ✅：经济产权（实际控制力）和法律产权（法律认定）是两个不同层面。
  • D 错误：权威与产权密切相关——谁拥有产权，谁就有权威。

【判断题】股权分散的现代大企业，所有者失去对管理层的权威。（）
答案：✅ 正确。

【判断题】不平等投票权可以解决股权分散的问题。（）
答案：✅ 正确。如阿里巴巴的合伙人制度、京东的AB股结构。

4.4 剩余控制权

完备合约 vs 不完备合约：

  • 完备合约：一切情况都事先约定好 → 无所谓产权，无所谓谁是老板
  • 不完备合约：总有未事先规定的情况 → 产权的本质 = 剩余控制权

核心逻辑：合约不可能完备 → 需要有人对未约定事项"拍板" → 拥有剩余控制权的人就是"老板"。

关键：产权或剩余索取权能填补合约的不完备性（不是"无法"）。

【单选题】关于剩余索取权，以下说法正确的是（）。
A、完备合约的分析框架下，剩余索取权更为重要。
B、对于不完备合约，一旦没有事先规定的情况发生，老板也不能"拍板"。
C、承包制下的"卖断"，老板是"拍板者"。
D、对于完备合约，无所谓产权，无所谓谁是老板 ✅

选项分析：

  • A 错误：完备合约下，一切已约定，剩余索取权不重要。正是不完备合约下，剩余索取权才重要。
  • B 错误：不完备合约下，正是因为有未约定的情况，老板可以拍板（剩余控制权）。
  • C 错误：承包制下"卖断"后，承包方是拍板者，不是原老板。
  • D ✅：完备合约 = 一切已约定 = 没有剩余 = 无所谓产权和老板。

【判断题】在不完备合约的分析框架下，产权的本质是剩余控制权。（）
答案：✅ 正确。

【判断题】产权或剩余索取权无法填补合约的不完备性。（）
答案：❌ 错误。产权和剩余索取权正是用来填补合约不完备性的机制。

4.5 机会主义与内部化

机会主义：经济主体为实现自身利益而采取的欺骗、隐瞒等策略性行为。

内部化理论（威廉姆森提出）：为了消除机会主义的影响，将外部交易内部化为企业内部行为。

FDI（外商直接投资）：外国企业直接在中国投资建厂，目的之一是避免技术资产流失（内部化的一种形式）。

【判断题】外国企业直接在中国投资建厂，是为了避免技术资产的流失。（）
答案：✅ 正确。直接投资建厂（绿地FDI）可以更好地控制核心技术，避免技术许可方式下的技术泄露。

【判断题】奥利弗·威廉姆森提出的内部化理论是为了消除机会主义的影响。（）
答案：✅ 正确。内部化通过将市场交易转化为企业内部指令，减少信息不对称和机会主义行为。

4.6 联合生产与监督

熊彼特的"三驾马车"：好的经济分析必须具备历史、理论、统计三个维度。

注意：不包括"问题"。题目中"问题"是干扰项。

联合生产：需要监督者来确定每个成员的责任与贡献。如果企业所有者充当监督者，监督的成本和收益就被**“内化”**了。

【多选题】奥地利经济学家熊彼特认为好的经济分析必须具备三个维度是（）。ACD
A、历史 ✅
B、问题
C、理论 ✅
D、统计 ✅

选项分析：

  • A 历史 ✅：任何问题只有在具体的历史背景下才能洞悉其重要性。
  • B 问题 ❌：干扰项，不是熊彼特三驾马车之一。
  • C 理论 ✅：没有理论的经济分析是肤浅的，缺乏洞见的。
  • D 统计 ✅：没有数据支撑的经济分析是缺乏科学性的。

记忆：熊彼特三驾马车 = 历史 + 理论 + 统计（不含"问题"）。

知识并非纯粹的公共品——知识带有公共品性质，但具有**“半公半私”**的特征。

6.4-6.6 李约瑟之谜

李约瑟之谜：为什么中国古代科技领先，却没有发生工业革命？

中国封建主义制约创新的原因：

  • 重本抑末限制了重商主义的发展
  • 官本位体制耗尽了精英的时间
  • 摒弃任何威胁帝国安全的技术，制度内卷化

注意：不是"人稠地狭劳动替代需求不强"——这是干扰项。

林毅夫的解释：

  • 早期技术创新由经验驱动（中国人口多 → 经验多 → 创新多 → 领先）
  • 科学革命后技术创新有了理论指导（理论创新需要系统性研究 → 中国没有科学传统 → 落后）
  • 在中国古代，技术创新有收益（不是"没有收益"）

技术创新的本质两难：

  • 技术的独占性 vs 技术价值的最大化（市场份额越大，控制权丧失的可能性越大）
  • 市场价值与市场范围成正比（不是"反比"）

B、官本位体制耗尽了精英的时间 ✅

7.5 互联网的"中心化"力量

先动优势 vs 后发优势：

  • “早起的鸟儿有虫吃” → 先动优势
  • “先下手为强，后下手遭殃” → 先动优势
  • “枪打出头鸟” → 后发优势（先动者风险大）
  • “逆水行舟，不进则退” → 无关（讲的是持续努力，不是先动/后发）

关键判断：

  • 互联网作为中心化力量 → 后来者进入成本很高（不是"不高"）
  • 网络外部性强化的是先动优势（不是后发优势）→ 所以赢者通吃
  • 先动优势强化"明星效应"，后来者居上更难

7.16 中美关系

传统国际分工：劳动相对丰裕的国家生产和出口劳动密集型产品（不是"技术密集型"）。

智能机器人的出现会打破国界线这一劳动力自由流动的边界。

【单选题】以下选项中，对于传统的国际分工与国际贸易的理解，有误的是（）。

B、资本自由流动，劳动跨国流动受限
C、劳动相对丰裕的国家生产和出口技术密集型产品 ✅（此选项有误）
D、不同国家的资本-劳动比不同

  • C ✅（此选项有误）：劳动丰裕的国家应生产和出口劳动密集型产品（比较优势理论），不是"技术密集型"。

【判断题】智能机器人的出现仍然不会打破国界线这一劳动力自由流动的边界。（）

马克思共产主义的三个条件：劳动生产率极大丰富、劳动成为第一需要、按需分配。

【判断题】人工智能时代的到来，会使得资本主义社会的根本矛盾自然消解。（）

第八章 制度与交易

8.1 制度的功能

Avinash Dixit 从功能角度把制度定义为"做蛋糕"与"分蛋糕"。

核心观点：

  • 分蛋糕的方式会影响做蛋糕的积极性
  • 好的制度是：切蛋糕的人最后拿蛋糕（不是"先拿"）
  • 自愿交易是双赢的、交易满足多样性、分工提高生产率

8.5-8.6 不完全信息与非理性

非理性的人不会积极去冒充成理性（因为非理性本身就是无法自控的）。

【判断题】在不完全信息交易下，卖家评价与买家评价都是他们的私人信息。（）

第九章 均衡与系统思维

9.1 勒沙特列原理

勒沙特列原理：任何系统面对任何冲击都会自动衍生出相应的对冲力量，以维护原有的系统。

经济系统中的供求规律：

  • 价格是市场交易的调节机制
  • 价格越高，需求越少，供给越多
  • 需求冲击让需求提高 → 供不应求 → 短期价格上升
  • 短期价格上升 → 供给增加 → 长期价格向下

【多选题】以下说法符合经济系统中的供求规律和均衡机制的是（）。ABCD
A、价格是市场交易的调节机制 ✅
B、价格越高，需求越少，供给越多 ✅
C、需求冲击让需求提高，则供不应求，短期价格随之上升 ✅
D、短期价格上升，导致供给增加，长期价格又随之向下 ✅

【判断题】勒沙特列原理是指任何系统面对任何冲击都会自动衍生出相应的对冲力量，以维护原有的系统。（）
答案：✅ 正确。

9.2 马尔萨斯陷阱

马尔萨斯陷阱：人口按几何级数增长，生活资源只能按算术级数增长 → 饥饿、战争和疾病。

马尔萨斯陷阱是勒沙特列原理的一种特殊应用。

【判断题】马尔萨斯陷阱，指当人口按几何级数增长，生活资源只能按算术级数增长，就会不可避免地导致饥饿、战争和疾病。（）
答案：✅ 正确。

【判断题】马尔萨斯陷阱是勒沙特列原理的一种特殊应用。（）
答案：✅ 正确。

9.3 卢卡斯批评

卢卡斯批评：由于人们的理性预期，各种相机抉择的政策最终会趋于无效。

它是勒沙特列原理的特殊应用。

关键辨析：如果人们能理性预期到政策者的"小九九"，政策执行力度将趋于最优化是错误的——应该是政策效果趋于无效（人们的行为调整抵消了政策效果）。

颠倒原理：很多政策或决策产生的动态效应与静态效应恰好相反。

【单选题】以下说法不符合卢卡斯批评的是（）。
A、它是勒沙特列原理的特殊应用
B、由于人们的理性预期，各种相机抉择的政策最终会趋于无效
C、如果人们能理性预期到政策者的小九九，政策执行力度将趋于最优化 ✅（此选项不符合）
D、政府制定的最优政策，都是基于当时的状态变量特定"参数"下的优化结果

选项分析：

  • C ✅（此选项不符合）：人们理性预期到政策后，政策效果趋于无效（不是"最优化"）。因为人们会提前调整行为来抵消政策效果。

【判断题】简单来说，颠倒原理是指很多政策或决策产生的动态效应与静态效应恰好相反。（）
答案：✅ 正确。

9.4 无处不在的倒U型

拉弗曲线：税率增加，税收不一定增加（超过最优点后，高税率抑制经济活动 → 税基缩小 → 税收反而下降）。

【判断题】拉弗曲线的含义是，税率增加，税收一定增加。（）
答案：❌ 错误。拉弗曲线说明税率与税收呈倒U型关系——税率超过最优点后，税收反而减少。
